推荐一款开源笔记本软件 Joplin

Posted on 2019-02-03

最近开始使用一款开源笔记本软件 Joplin，感觉挺好用的，推荐一下。我之前使用过 evernote、onenote、wiznote、有道云笔记以及国内的印象笔记，我将通过对比这几款笔记本软件的优缺点，来说说我的推荐理由。

首先，每款笔记都有各自对应的用户群体和市场，用户各取所需。我推荐 Joplin 是因为这款软件是开源软件而且正好符合我需要的一些特性，如

  1. 有标签

  2. 支持 markdown，且可以用外部编辑器编辑，搭配 Typora + PicGo（图床工具）使用很舒服

    要求 markdown 的原因主要是因为要保存代码以及图片可以保存在图床上，纯文字占用空间小。

  3. 跨平台，window，Android，ios，mac，chrome 剪藏插件都有

  4. 自己持有数据，方便数据迁移，同步可选 Dropbox、OneDrive、WebDAV、本地

  5. 数据可加密保存至远端

  6. 免费或者可以自己组装各种服务。我不是很喜欢一键订阅制，因为有些特性我愿意付费，有些特性我不想为之付费的

以下两款笔记软件不参与对比：

  1. 为知笔记

    我高中的时候主要用的就是为知笔记了，简约风格吧，不过免费会员无法同步政策出来之后，我就没用了。

    长时间不用会被 tag 成非活跃用户，有数据丢失的风险。

  2. Standard Note

    需要购买插件，不支持外部 editor 打开文件。

1、标签功能

软件 是否支持
OneNote 不支持，可以通过 #标签# 来表示标签，不过中文搜索贼烂
Evernote 支持
有道云笔记 支持
Joplin 支持

2、markdown

软件 是否支持
OneNote 不支持，图片无法通过外链插入
Evernote 不支持，图片无法存储在图床上，放大了流量上传的限制；国内印象笔记支持，不过目前功能好像不太完善，切换 md 文件默认编辑模式这点很不舒服。
有道云笔记 支持，样式有点丑
Joplin 支持，可以通过外接编辑器编辑 markdown。我一般会采用 Typora + PicGo 写笔记，直接保存即可，不用将文本复制到笔记本中（有道笔记本需要）。并且，可以自定义 markdown 的渲染样式。

3、搜索功能

软件 是否支持
OneNote 支持，中文搜索巨弱👎
Evernote 支持，好用，搭配 Alfred 也好用
有道云笔记 支持标签和中文搜索，软件内搜索
Joplin 支持标签和中文搜索，且由于数据自持，可以将文件存储在电脑中（我是 mac），通过 spotlight 和 alfred 等外部软件实现更好的搜索体验（正则表达式之类的）。

4、多平台及上传空间

软件 是否支持
OneNote 支持同步，大厂保证，覆盖全平台，同步体验良好，微软倒闭、数据丢失风险低。
不支持本地使用，无法停止同步
Evernote 支持同步，非会员 60 M 上传限制，且只能同步两个平台。
不支持本地使用，无法停止同步
有道云笔记 支持同步，3 GB 免费
不支持本地使用，无法停止同步
Joplin 支持同步，自购云服务（OneDrive、Dropbox、WebDEV 或者本地）。好处在于云服务同时可以用于其他的软件，而不是新买一个软件就要新买一个云存储服务。

5、网页剪藏

都支持，Evernote 最好用。Joplin 由于只支持 markdown，所以网页的存储可能存在兼容转化问题。

6、数据迁移

基本上数据自持的话，数据迁移会比较方便一些。好像大部分厂商都不支持批量导出。

  1. OneNote

    好像只能导出 pdf

  2. Joplin 可以导出多种格式，并且由于可以将本地设置成远端，利用同步也可以随时增量导出文件。

  3. 有道云笔记支持下载 markdown，导出 pdf，免费会员无法批量导出。

7、历史版本

软件 是否支持
OneNote 支持。
Evernote 支持。
有道云笔记 支持，好像有限制。
Joplin 不支持，不过可以通过云端提供的记录历史版本的服务（如 onedrive）或者 git 来解决，不过我一般没这个需求。

8、数据端到端加密

Joplin 支持，其他不清楚，因为数据都是存储在服务提供方。

9、分享发布

国内笔记软件的优势，可以通过链接分享至微信等平台，Joplin 想要分享的话只能自己搭服务器了。

补充

  1. evernote 插件丰富，web clipper 贼好用，和笔记搭配的一些应用也会优先支持和 evernote 进行交互。

  2. OneNote 很适合学生上课做笔记，我大学课上基本就用的是 OneNote 记笔记。

    优点：

    1. 排版自由，支持手写。这个贼强，上课的时候如果涉及画图、树、网络之类的逻辑很好用。
    2. 多人协作好用，有道云协作要会员。不过我一般用 Google Doc。

    缺点：

    1. 不支持 markdown
    2. 国内同步可能会有坑
    3. 无法导出
    4. 搜中文巨坑
    5. 没有标签
    6. 空间 5 GB

    OneNote 的好与坏对比鲜明！！！

  3. Joplin 好像是基于 Electron 开发的，有能力的可以自己开个分支添加 feature。
